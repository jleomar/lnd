HTML-PRACTICA 1

Esta práctica es una aplicación de muchos elementos de HTML vistos en clase. Utiliza listas, tablas, imágenes, links a páginas http, etcétera, ... Este texto está justificado.


Los alumnos deben tener en cuenta:

  1. Es obligatoria para todos los alumnos que tengan el módulo MEY, sepan o no sepan crear documetnos en HTML con anterioridad
  2. Utilicen la documentación que les proporciona la profesora para confeccionarla
  3. En cuanto a los recursos:
    1. Las fotos pueden cogerlas
      • de la red
      • de su propio equipo
    2. Deben respetar el texto y los tipos de objetos, los tamaños hacerlos aproximados
    3. Este texto también es obligatorio
  4. La nota de esta práctica se tendrá en cuenta en la evaluación
